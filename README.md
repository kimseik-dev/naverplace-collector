# 📍 네이버 플레이스 수집기

키워드로 네이버 플레이스 검색 결과(업체명·주소·전화·리뷰수·플레이스 URL·광고여부 등)를 한 번에 수집해 엑셀로 저장하는 **웹 대시보드**입니다.

> ⚠️ **학습/연구 목적으로만 사용해주세요.** 네이버 이용약관과 robots.txt를 준수하고 과도한 요청은 자제해주세요.

---

## 💎 단일 실행파일로 받기 (가장 쉬움)

GitHub Release 페이지에서 OS 에 맞는 zip 을 다운로드합니다.

| OS | 파일 |
|----|------|
| Mac | `NaverPlace-mac.zip` |
| Windows | `NaverPlace-win.zip` |

압축 풀고:
- **Mac**: `실행하기.command` 더블클릭
- **Windows**: `NaverPlace.exe` 더블클릭

Python 설치도, 명령줄 사용도 **불필요**합니다. 잠시 후 브라우저가 자동으로 열려요.

> 🛡 **첫 실행 시 보안 경고**
> - Mac: 우클릭 → 열기 → 다시 한 번 "열기"
> - Windows: "추가 정보" → "실행"
> ※ 정식 코드사이닝이 안 되어 있어서 그렇습니다 (악성 코드 아닙니다).

## 🚀 소스에서 직접 실행 (개발자용)

### 1단계 — Python 설치 (최초 1회만)

이미 설치되어 있다면 건너뛰세요.

- **Mac**: <https://www.python.org/downloads/macos/> 에서 최신 버전 다운로드 후 설치
- **Windows**: <https://www.python.org/downloads/windows/> 에서 다운로드. 설치 시 **"Add Python to PATH"** 옵션을 반드시 체크!

> Python **3.10 이상**이 필요합니다.

### 2단계 — 폴더 받기

이 폴더 전체를 원하는 위치(예: 바탕화면)에 복사하세요.

### 3단계 — 설치 (최초 1회만)

- **Mac**: `install_mac.command` 파일을 **더블클릭**
- **Windows**: `install_windows.bat` 파일을 **더블클릭**

> Mac에서 "확인되지 않은 개발자" 경고가 뜨면: 파일을 우클릭 → "열기" → 다시 한 번 "열기"를 누르세요.

설치 진행: Python 패키지 다운로드(1~2분) + 브라우저 다운로드(2~3분).

### 4단계 — 앱 실행

- **Mac**: `run_mac.command` 더블클릭
- **Windows**: `run_windows.bat` 더블클릭

→ 잠시 후 **브라우저가 자동으로 열리며** 웹 대시보드가 표시됩니다.

### 5단계 — 사용

1. 좌측 사이드바에서 **최대 수집 개수**와 **카테고리**(자동 추천) 선택
2. 상단에 **검색 키워드** 입력 (예: `강남맛집`, `강남미용실`)
3. **🔍 수집 시작** 버튼 클릭
4. 진행 상황 실시간 확인 (1~5분 소요)
5. 완료되면 **📥 엑셀 파일 다운로드** 버튼 클릭

> 종료할 때는 실행 창에서 `Ctrl+C` 를 누르거나 그냥 창을 닫으면 됩니다.

## 💡 사용 팁

### 카테고리 선택
- "자동 추론"이면 키워드에서 카테고리를 알아냅니다 (맛집→음식점, 미용실→헤어샵 등)
- 자동이 안 맞으면 사이드바에서 직접 선택하세요

### 수집 개수 한계
- 네이버 플레이스는 **키워드별로 최대 노출 결과 수에 제한**이 있습니다
- "강남맛집"·"강남미용실"의 경우 **약 60건**까지 제공
- `최대 수집 개수`를 100으로 잡아도 네이버 한계까지만 모입니다

### 차단 회피
- 너무 자주 연속 실행하면 일시 차단됩니다 → **1~2분 대기 후 재시도**

## 🛠 명령줄 사용 (고급)

웹 대시보드 대신 CLI 로도 쓸 수 있습니다.

```bash
# Mac
source .venv/bin/activate
python main.py 강남맛집 --limit 100

# Windows
.venv\Scripts\activate
python main.py 강남맛집 --limit 100
```

추가 옵션:
- `--limit 100` : 최대 수집 개수
- `--category restaurant` : 카테고리 강제 지정
- `--no-enrich-phones` : 전화 보강 끔 (더 빠름)
- `--no-headless` : 브라우저 창 표시 (디버깅)

```
├── src/
│   ├── scraper.py           # 스크래퍼 코어 (GraphQL 인터셉트)

│   └── exporter.py          # 엑셀 저장
│
└── output/                  # 수집 결과 저장 (자동 생성)
```

## ❓ 문제 해결

| 증상 | 해결 방법 |
|------|----------|
| Mac에서 더블클릭 시 "확인되지 않은 개발자" | 우클릭 → 열기 → 다시 한 번 열기 |
| `Python 3.10 이상이 필요` 메시지 | <https://python.org> 에서 최신 Python 설치 |
| 브라우저가 자동으로 안 열림 | <http://localhost:8501> 직접 접속 |
| 카드 로드 타임아웃 | 1~2분 대기 후 재시도 (일시 차단됨) |
| 수집 0건 | 네이버가 클래스명을 변경했을 수 있음 → `src/scraper.py` 의 `li.UEzoS, li.p0FrU` 부분 점검 |
| 미용실은 전화가 비어있음 | 사이드바에서 "전화번호 자동 보강" 체크되어 있는지 확인 |
